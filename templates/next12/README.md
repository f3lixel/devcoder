# Sandpack React Starter

Dieses Template ist auf das integrierte Sandpack-Preview ausgelegt. Es enthält eine kleine React-18-Anwendung
(ohne Next.js oder Server-Code), damit alles vollständig im Browser kompiliert werden kann.

## Struktur

- `src/App.jsx` – Beispiel-Komponente
- `src/main.jsx` – Entry-Point mit `createRoot`
- `styles.css` – Dark-Theme Styling
- `public/index.html` – Mount-Point

## Nutzung

1. Dateien können beliebig erweitert werden (`src`-Ordner, zusätzliche Komponenten etc.).
2. Sandpack kompiliert automatisch nach jedem Speichern.
3. Für persistente Speicherung sorgt weiterhin die Files-API des Projekts.
